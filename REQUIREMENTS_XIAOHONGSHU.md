## 小红书图片生成需求

### 功能范围
* **功能同 v3.0**：
  * 风格选择 (Linear / Knowledge)
  * 4张卡片生成与预览
  * 单张下载与批量下载
  * 支持上传封面图，未上传时使用默认封面图
  * 移动端显示长按保存提示，下载按钮可能不生效
  * 局域网手机测试可通过 URL 参数覆盖后端地址
  * 局域网手机测试允许后端返回局域网 Origin
* **尺寸**：1080 × 1440 (PNG格式)
* **背景留白**：左右 120px，上下 140px
* **安全区域**：内容区域最大宽度 840px

### 风格系统详解

**1. Linear 风格 (极简产品风)**

* **定位**：极简、产品说明书、方法论
* **背景**：#FAFAFA
* **文字**：主文字 #111111，正文 #555555，强调色 #4F7FFF
* **字体**：中文 Noto Sans SC，日文 Noto Sans JP
* **字号**：标题 64px，副标题 42px，正文 36px，日语句子 38px，说明 28px，标签 24px
* **特殊元素**：
  * 左侧竖线：宽度 6px，颜色 #4F7FFF
  * 标签样式：背景 #EEF3FF，文字 #4F7FFF，圆角 12px

**2. Knowledge 风格 (知识卡片风)**

* **定位**：知识博主、学习卡片、易传播
* **背景**：#FFFFFF (辅助背景 #F6F6F6)
* **文字**：#222222，强调色 #FF7A45
* **卡片容器**：圆角 24px，阴影 10%，背景 #FFFFFF

### 排版规则

* **日语句子**：自动换行，行宽最大 18 字符
* **避免**：超长行
* **间距**：段落间距 24px，模块间距 48px

#### 卡片1（封面）

数据来源：

```
分享标题
```

**展示规则**：
*   **左上角标签**：显示 "日语表达升级练习 Day X"
    *   X 为当前日期距离 2026年3月6日 的天数（含当天，即3月6日为Day 1）
*   **背景图**：卡片中间区域放置正方形背景图
    *   支持上传封面图，未上传时使用默认封面图
    *   静态图片目录：`public/assets/xiaohongshu/`
    *   默认图片文件名：`cover-default.png`
    *   布局：正方形居中，等比缩放适配（contain），不裁剪、不拉伸，留白填充
*   **主标题**：来源于"分享标题"，大字显示，位于背景图下方
*   **右下角署名**：修改为 "乐爸学日语"

示例：

```
日语表达升级练习 Day 23

日语表达“最近一直在思考”
```
